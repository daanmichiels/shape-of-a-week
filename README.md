# shape-of-a-week

> _If you had to put seven points on a plane, each corresponding to a day of the week,
> what would be a sensible way of doing that?_

This is what we try to answer, based on commit histories of GitHub repositories.

Get the IPython notebook, or visit
[http://www.daanmichiels.com/shape-of-a-week/](http://www.daanmichiels.com/shape-of-a-week/)
for a pre-rendered version in your browser.
